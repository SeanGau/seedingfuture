# 檔案結構
* 程式使用python3 flask做框架
* 來源圖片放置於 static/source
* 背景圖片放置於 static/source/bg
* 程式於執行時會自動產生thumb縮圖

# how to run
1. 建立python3 virtual enviroment
2. 設置好DATABASE_URL環境變數
3.
```bash
pip3 install -r requirement.txt
python3 app.py
```
